Graphy Application Reference Guide

Graphy is an Android Application written by Shane Taylor and developed by Shane Taylor, Daniel Fitch, Itzel Ruiz, Joseph Hibdon, and Rachel Adler, all of Northeastern Illinois University.  Graphy reinforces the basics of the cartesian plane and properties of triangles through interactive screens that present problems to users for solving.

From the main menu screen, the user can click an image to jump directly to any of the six activities.

The truth table screen is a brief introduction to formal logic and truth tables with a “If P then Q” table and related text example.
